XmlTypeInfoProvider

implements XMLizable
com.sforce.ws.bind.XmlTypeInfoProvider

Class Overview

This interface can be used to indicate that it knows what its type Info is, and to use this in preference to asking to the TypeMapper for the type info. This is primarily used by XmlObject. If you want to defer to the default type mapping, you can return null from the getTypeInfo call (or just not implement this interface)
