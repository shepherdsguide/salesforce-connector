com.sforce.ws.wsdl

Provides an WSDL Object model that can parse and generate WSDL.

To parse a WSDL:

    String wsdlFile = //location of the wsdl
    Definitions d = WsdlFactory.create(wsdlFile);

Package Specification

  • Simple Object Access Protocol (SOAP) 1.1
  • XML Schema
